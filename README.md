## Overview

Events generates hundreds or even thousands of photos captured across different devices. Whether it's a wedding, graduation, birthday, corporate event, or vacation, finding the best memories later becomes difficult and overwelming.

**PicsDrop AI** automatically analyzes uploaded event photos using **Google Gemini Vision** and organizes them into an intelligent photo collection.

Instead of repeatedly analyzing every image for different features, PicsDrop AI follows a **single-analysis, multi-agent architecture**. Each uploaded image is analyzed once using Gemini Vision, producing a structured `analysis.json` that becomes the shared knowledge source for all downstream agents.

This architecture minimizes repeated AI inference while allowing specialized agents to collaborate on different tasks.

# Multi-Agent Workflow

## 1. Upload Photos

Users upload event photos through the React frontend.

The FastAPI backend stores uploaded images locally for processing.

---

## 2. Vision Analysis

The **Vision Analysis Agent** sends each uploaded image to **Google Gemini Vision**.

Gemini performs a comprehensive image analysis and generates structured metadata describing:

- Scene
- Objects
- Semantic tags
- Generated caption
- Lighting
- Blur detection
- Composition
- Event context
- People information

This information is stored as:

```
analysis.json
```

Each image is analyzed **only once**.

---

## 3. Quality Scoring Agent

The **Quality Scoring Agent** consumes the structured analysis generated by Gemini and assigns an overall quality score using custom scoring rules based on:

- Blur
- Lighting
- Composition
- Image visibility
- Overall quality

---

## 4. Caption Agent

The Caption Agent standardizes and exposes the natural-language description generated during the Vision Analysis stage for downstream use. Future work will extend this agent to generate social media captions, hashtags, alt text, and personalized caption styles.

---

## 5. Duplicate Detection Agent

Duplicate detection uses **perceptual image hashing (ImageHash)**.

Each uploaded image receives an image hash, allowing visually similar photos to be identified without relying on filenames.

When duplicates are detected, the highest-quality image is recommended.

---

## 6. Smart Album Agent

The Smart Album Agent automatically groups related images using semantic metadata generated during vision analysis.

Albums are created based on:

- Scene
- Event context
- Objects
- Semantic tags

No manual folder creation is required.

---

## 7. Reel Planner Agent

The **ReelPlannerAgent** creates an AI-generated highlight reel plan for the entire photo collection.

Rather than processing images again, it consumes the structured metadata generated during the initial image analysis. Before invoking Gemini, the agent filters the collection by selecting the highest-quality, non-duplicate photos, ensuring the AI focuses only on the strongest moments from the event.

Using these curated photo descriptions together with a reusable **Agent Skill**, the ReelPlannerAgent generates:

- рџЋ¬ Reel title
- рџ“– Story arc
- рџ“· Suggested photo sequence
- рџ’¬ Overlay text for each scene
- рџЋµ Music mood recommendation
- вњЁ Transition suggestions
- рџЏЃ Closing caption

Because the agent receives only structured photo metadataвЂ”not the original imagesвЂ”it avoids repeated image analysis, reduces API usage, and performs collection-level reasoning efficiently while producing a cohesive event highlight reel.

---

## 8. Knowledge Retrieval

The **Orchestrator Agent** also performs knowledge retrieval.

When users ask natural-language questions, the Orchestrator searches the stored `analysis.json` data across the collection and returns matching photos.

Example queries:

- Show me wedding photos
- Find duplicate images
- Show outdoor group photos
- Which photos have the highest quality score?

# Design Philosophy

PicsDrop AI follows a modular multi-agent architecture where each agent has a specialized responsibility.

The ImageAnalysisAgent performs a single Gemini Vision analysis for each uploaded image and stores structured knowledge in `analysis.json`.

Subsequent agents reuse this shared knowledge instead of repeatedly analyzing images, reducing API usage while keeping the system modular.

The ReelPlannerAgent demonstrates higher-level reasoning by combining collection metadata with reusable Agent Skills to generate an AI-planned event highlight reel.
